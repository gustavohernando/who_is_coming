Productivization Project – (Project name) This is a Data Science project based on a PoC (proof of concept). There are three main parts that constitutes the project. Those parts are: ETL with Spark, modeling and Databricks´s cloud analysis, and creating an app and uploading it in a cloud´s platform.

First part, ETL with Spark
Dataset found and used is about medical appointments. The selected team for this part has worked with the data. They have treated and cleaned the data using Spark (outliers, nulls, making new tables, etc). Feature engineering is being also applied.

Second part, modeling and Databricks´s cloud analysis
Dataset is cleaned and ready to use. The selected team has worked to implement a machine learning model that tries to predict the assistance or not assistance of the medical appointments. For this purpose, they have used Pandas, SKlearn among others. Finally, they have also made an analysis of Databricks´s cloud too.

Third part, creating an app and uploading it in a cloud´s platform
Machine learning model is ready to use. By using Flask, the selected team has created an app. This app will contain the model. Tryouts have been made in local to test the app. Once the app worked correctly, the team launched the app in PythonAnywhere. It is just not local, now everyone can have access to it.
